# infoNest Backend

## Overview

This repository is intended to contain the backend of the **infoNest** project. It is planned to be built with **Node.js** and **Express.js**, and will provide core API services for the frontend.

Planned responsibilities include:

- Handling user authentication and authorization
- Managing data storage and retrieval
- Serving API endpoints for the frontend
- Implementing basic security and input validation

This is an initial setup, and the implementation details may evolve as the project progresses.

## Technology Stack

- **Node.js**
- **Express.js**
- **Database: NeonDB - PostgreSQL**
- RESTful API design
- Basic security measures
